## UI 구현 원칙

> **디자인 기준**: `reference/claude-design/` 프로토타입 + `styles/token.css` 토큰
>
> - **페이지 배경**: `--color-neutral-150` (연한 회색) — 흰색 사용 금지
> - **카드**: 흰 배경 + 보더 없음 + `shadow-1` 항상 표시 (`border-radius: var(--radius-2xl)`)
> - **선택 칩**: `bg-fg-strong` (검정) + 흰 텍스트 — 파란색(`accent`) 사용 금지
> - **비선택 칩**: `neutral-200` 회색 배경 + 보더 없음
> - **캘린더 선택 날짜**: 검정 원 (`bg-fg-strong`) — 파란 원 사용 금지
> - **AppBar / sticky 헤더**: 페이지 배경색과 동일 (회색), 하단 보더 없음
> - **탭 바**: 흰 배경, 위쪽 보더 대신 `box-shadow: 0 -1px 0 subtle`
> - **모달 / 바텀시트**: `border-radius: var(--radius-3xl)` (32px)
> - **인풋**: `border-radius: var(--radius-sm)` (8px), 보더 유지
> - **액센트 색상**: `--color-accent` (#0066FF) — 인터랙티브 강조(링크, 포커스 등)에만 한정
